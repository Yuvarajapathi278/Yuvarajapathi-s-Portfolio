## 🎨 Customization Guide

### Colors (Easy!)
Edit `tailwind.config.ts`:
```typescript
colors: {
  'primary': '#0f172a',      // Dark background
  'secondary': '#1e293b',    // Card background
  'accent': '#3b82f6',       // Button blue
  'neon': '#00ffff',         // Highlight cyan
}
```

### Add Projects
Edit `src/components/Projects.tsx`:
```typescript
const projects = [
  {
    id: 1,
    title: 'Your Project',
    description: 'What it does',
    tech: ['React', 'TypeScript'],
    demo: 'https://demo-url.com',
    github: 'https://github-url.com',
  }
]
```

### Update Skills
Edit `src/components/Experience.tsx`:
```typescript
{
  category: 'Languages',
  skills: ['Java', 'JavaScript', 'TypeScript']
}
```
